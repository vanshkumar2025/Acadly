# Acadly — Website

Acadly is a multi-portal academic web app (College, Teacher, Student) designed for live attendance, announcements, timetables and progress tracking.

Quick overview
- Role-based portals: /college, /teacher, /student
- Live attendance with QR sessions (simulated)
- Timetable and announcements per role
- Student attendance progress and productivity widgets
